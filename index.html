Week-by-Week Curriculum

9 weeks. 27 classes. 54 hours.
Practical. Outcome-driven.

Week 01

AI Foundations & n8n Automation

  • Agentic AI vs chatbots — planning, memory, tool use
  • Model Context Protocol (MCP) — the 2026 agent stack
  • Build your first AI workflow in n8n
  • Multi-agent orchestration patterns with Zapier + n8n
Week 02

n8n Intermediate — Data & APIs

  • Advanced data manipulation with Claude + Code node
  • REST APIs, OAuth, webhooks & paginated responses
  • Sub-workflows, LLM chains & agentic loops
  • Claude tool_use API + state management with Supabase
Week 03

n8n Advanced — Production & RAG

  • Self-hosted n8n on Docker with Postgres & Redis
  • RAG pipeline: ingest → chunk → embed → retrieve → generate
  • MCP servers as agent tools with persistent memory
  • Real project library: Hotel Bot, Real Estate Qualifier & more
Week 04

Voice Agents — Vapi & ElevenLabs

  • ASR → LLM → TTS pipeline & latency optimisation
  • ElevenLabs voice cloning for branded agents
  • BANT lead qualification + Google Calendar booking
  • Twilio outbound campaigns, SMS & TCPA compliance
Week 05

Vibe Coding — Lovable & Claude Code

  • Build full-stack SaaS apps with Lovable + Supabase
  • Claude Code CLI — agentic coding in the terminal
  • Cursor Composer — edit multiple files simultaneously
  • Deploy to Vercel, Railway & Netlify — client handoff SOPs
Week 06

Ecommerce + Meta Ads

  • Shopify AI store — product copy, chatbot & order automations
  • Meta Ads — ROAS, CTR, lookalike audiences explained
  • AI-generated ad creatives at scale with Claude + Canva AI
  • Auto-pause bot using Meta Ads API + n8n + Claude analysis
Week 07

Client Acquisition

  • Automated lead research — 50 leads in 20 min with Apollo.io
  • 5-touch cold email + Vapi warm call outreach sequence
  • Discovery calls, SPIN questions & proposal building with Claude
  • Pricing retainers, handling objections & closing scripts
Week 08 ✦ NEW

Personal Branding & Content Authority

  • LinkedIn optimisation + YouTube channel setup for AI experts
  • Automated content engine — 1 idea → 10 pieces via Claude + n8n
  • Case studies, testimonials & inbound lead funnel
  • 5-email nurture sequence → booked discovery calls on autopilot
Week 09

Freelancing & Agency Capstone

  • Build a full agency automation system worth $2,000/month retainer
  • Connect n8n, RAG agents, Vapi, Notion CRM & Slack in one system
  • Client handoff: exported JSONs, screen recordings & credentials vault
  • Pricing proposals, Stripe setup, Upwork positioning & agency OS
